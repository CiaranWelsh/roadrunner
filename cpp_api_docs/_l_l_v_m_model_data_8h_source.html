23 /**
24  * A data structure that is that allows data to be exchanged
25  * with running SBML models. In the case of CExecutableModels, A pointer to
26  * this struct is given to the compiled shared library, and the C code
27  * there modifies the buffers of this structure.
28  *
29  * There are some functions in ExecutableModel.h that manage ModelData
30  * memory. These would have made more sense here, but in order to prevent
31  * any issues with generated code interacting with them, they were placed
32  * there.
33  *
34  * Basic Nomencalture
35  * Compartments: A well stirred compartment which contains one or more species.
36  * the volume of a compartment can change durring the course of a simulation.
37  *
38  * Floating Species: these are chemical species who's values (ammount / concentration)
39  * change over time.
40  *
41  * Boundary Species: chemical species that who's values are fixed to their initial conditions,
42  * these function as boundary conditions.
43  *
44  * @see ExecutableModel.h
45  */
